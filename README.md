## 📊 Overview
The **Banking Customers Insights Dashboard** transforms raw banking data into actionable insights using **Advanced Excel** and **Microsoft Power BI**.  
The dataset (3,000 rows from Kaggle) was cleaned in Excel and transformed using **Power Query Editor (PQE)** for ETL processing.

---

## 🛠 Tools & Technologies
- **Advanced Excel** → Data cleaning, preprocessing, formatting  
- **Microsoft Power BI** → Dashboard creation, KPI visualization  
- **Power Query Editor (PQE)** → ETL pipeline for structured transformation  

---

## ❓ Problem Statement
Banks face challenges in identifying customer behavior patterns and predicting risks due to scattered data.  
This project addresses:  
- Customer segmentation based on demographics and financial activity  
- Tracking KPIs like loan defaults, deposits, and profitability  
- Visualizing customer insights for better decision-making  

---

## 🎯 Importance
- Provides **data-driven insights** into customer behavior  
- Helps reduce risk and improve customer retention  
- Enables strategic decision-making with clear KPI tracking  
- Demonstrates the synergy of **Excel + Power BI**  

---

## 🧹 Data Preparation
1. **Excel Cleaning**  
   - Removed duplicates and nulls  
   - Standardized formats (dates, currency, categories)  
   - Applied conditional formatting for error detection  

2. **Power BI Transformation (PQE)**  
   - Built ETL pipeline  
   - Created calculated columns & measures (Loan Default %, Profit Margin, Segmentation KPIs)  
   - Established relationships for interactive analysis  

---

## 📈 Dashboard Features
- **KPI Cards** → Total Customers, Loan Default %, Average Balance  
- **Charts** → Segmentation by Age, Income, Loan Status  
- **Matrix/Table** → Detailed breakdown of customer attributes  
- **Interactive Filters** → Region, Gender, Account Type
